# PRODUCT_OVERVIEW

**Product:** APForce\
**Version:** 1.0\
**Status:** Active

## Executive Summary

APForce is an AI-first customer engagement platform focused on WhatsApp
Business. It combines CRM, Inbox, Campaigns, Automation, Customer
Management, and Analytics into a single SaaS platform.

## Vision

Build the simplest enterprise-grade customer engagement platform for
SMBs.

## Target Customers

-   Financial advisors
-   Stock brokers
-   Insurance agents
-   Loan consultants
-   Real estate businesses
-   SMEs using WhatsApp as a primary sales channel

## Core Problems Solved

-   Scattered customer data
-   Manual follow-ups
-   Repetitive WhatsApp communication
-   Poor lead tracking
-   Lack of automation
-   No centralized CRM

## Core Modules

-   CRM
-   Contacts
-   Inbox
-   Templates
-   Campaigns
-   Automation
-   Employees
-   Customer Identity
-   Settings
-   Analytics (planned)
-   AI (planned)

## Technology Stack

Frontend: - Next.js - React - TypeScript - Tailwind CSS

Backend: - AWS Lambda - Express - DynamoDB - S3

Infrastructure: - Vercel - AWS - GitHub Actions

Messaging: - WhatsApp Business Platform (Meta)
